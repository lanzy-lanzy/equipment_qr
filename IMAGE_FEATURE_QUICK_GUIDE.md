# Quick Guide: Product Images Feature

## What's New
Supply items in your inventory can now have product images. See what you're managing at a glance!

## For Administrators & GSO Staff

### Adding Images to New Supplies
1. Go to **Supply Inventory** → **Add New Supply**
2. Fill in the basic information (name, description, category)
3. Scroll down to **Product Image** section
4. Click the file input area
5. Select an image file from your computer (JPG, PNG, GIF, or WebP)
6. See the preview appear with file size
7. Click **Create Supply** to save

**Tip**: File must be under 5MB. If image is too large, you'll see an alert.

### Updating Images on Existing Supplies
1. Go to **Supply Inventory** → Select a supply
2. Click **Edit** button
3. Scroll to **Product Image** section
4. Options:
   - **Upload New**: Click file input to select new image
   - **Remove Current**: Check the "Remove image" checkbox to delete existing image
5. Click **Edit Supply** to save

### Viewing Images
- **List View**: See thumbnail images next to supply names
- **Detail View**: View full-size image in the right sidebar with download button
- **Download**: Use download button to save image locally

## For Department Users

### Viewing Images
- See product thumbnails in the supply list
- View full images on supply detail pages
- Download images for your records

### Why Images Help
- Quickly identify what you're requesting or borrowing
- Reference what the actual product looks like
- Better documentation for your supply requests

## Image Requirements

| Requirement | Details |
|---|---|
| **Formats** | JPG, PNG, GIF, WebP |
| **Max Size** | 5 MB |
| **Recommended Size** | 1000×1000px (will resize automatically) |
| **Optional** | Images are completely optional |

## Tips & Best Practices

### Good Images
- ✅ Clear, well-lit photos
- ✅ Product centered in frame
- ✅ Plain background (white or light)
- ✅ Shows the actual item
- ✅ Good lighting/contrast

### Poor Images
- ❌ Blurry or dark photos
- ❌ Generic placeholder images
- ❌ Wrong item
- ❌ Very large files (>5MB)

### Example Supplies & Recommended Images
- **USB Flash Drive**: Close-up of the drive
- **Office Chair**: Side/front view showing the chair
- **Paper (500 sheets)**: Photo of the ream box
- **Printer**: Photo of the actual printer model
- **Pen Set**: Photo of the pen box or collection

## FAQ

**Q: Can I upload multiple images per supply?**
A: Currently, only one image per supply. Future versions may support image galleries.

**Q: What if the image is too large?**
A: You'll see an alert. Resize the image and try again. Most image apps can resize easily.

**Q: Can I crop or edit images?**
A: Crop images in your image editor before uploading. Future versions may include built-in cropping.

**Q: Where are images stored?**
A: In the system's media folder under `supply_images/`. You can also download images anytime.

**Q: Can I delete an image?**
A: Yes, when editing the supply, check "Remove image" to delete. The supply will show a default icon instead.

**Q: Are there watermarks or size limitations on display?**
A: Images display at their full resolution in the detail view. Thumbnails are automatically sized.

**Q: Can I batch upload images?**
A: Not yet, but you can add images one by one when creating or editing supplies.

## Troubleshooting

### Image won't upload
- Check file size (must be under 5MB)
- Verify file format (JPG, PNG, GIF, or WebP)
- Try a different browser
- Clear browser cache and try again

### Image displays incorrectly
- Image may be corrupted - try re-uploading
- Check browser zoom level
- Try a different browser
- Clear browser cache

### Can't see preview
- Make sure JavaScript is enabled in your browser
- Try a different browser
- Refresh the page

### Image is blurry/low quality
- Upload a higher resolution image
- Check if image was properly saved (not corrupted)
- Original image quality affects display

## Getting Help
